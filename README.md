Main gauche : Menu avec 3 boutons "Cube","Sphère" et "Cylindre"

Cliquer sur un des boutons avec la main droite puis cliquer dans l'espace fais apparaitre l'objet 3D (Appuyer plusieurs fois fais apparaître plusieurs objets)

Main droite : Menu avec 3 Sliders pour modifier la physique (Taille, Rotation et Position) de l'objet tenu dans la main droite.
